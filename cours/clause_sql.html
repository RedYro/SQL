Requêtes de base

  • Créer une BDD : CREATE DATABASE IF NOT EXISTS nomBdd
  • Supprimer une BDD : DROP DATABASE IF EXISTS nomBdd
  • Créer une table : CREATE TABLE nomTable (id INT KEY NOT NULL AUTO_INCREMENT, nom VARCHAR(100), prenom VARCHAR(100), email VARCHAR(255), birthday DATE, ville VARCHAR(200), code_postal VARCHAR(10), nombre_achat INT)
  • Insérer dans une table :

    INSERT INTO nomTable(id, nom, prenom, email, birthday, ville, code_postal, nombre_achat) VALUES (NULL, 'Lyfoung', 'Grégory', 'gregory.lyfoung@colombbus.org', '2001-03-31', 'Évry', '91000', '10')

    INSERT INTO users SET id_users = NULL , first_name = 'Thao' , last_name = 'Lau', email = 'thao.lau@colombbus.org' , birth_date = '2000-03-31' , city = 'Évry' , post_code = '91000' , purchase_number = '20';

  • Afficher le contenu d'une table : SELECT * FROM 'nomTable'
  • Effacer un enregistrement : DELETE FROM nomTable WHERE nom = ""
  • Effacer une table : DROP TABLE `nomTable`
